# Canonical Category List

This file is the single source of truth for allowed categories.
**No new categories should be added without updating this file.**
New concepts must enter as `tags` first and may only be promoted to a category
once at least 5 posts justify it.

---

## Layer A — Post type (exactly 1 per post)

| Category | Description |
|---|---|
| `Annotated Bibliography` | Structured summary and critique of a single work |
| `Descriptive Note` | Reading/study notes, outlines, or chapter summaries |
| `Essay` | Original analytical or argumentative piece |

## Layer B — Substantive theme (1–3 per post, from this list only)

| Category | Description |
|---|---|
| `Political Economy` | Relationship between economy and politics *within* a single context |
| `Comparative Political Economy` | Systematic *cross-country* or *cross-regime* comparison of political economies |
| `Politics of Education` | Education policy, actors, and political disputes |
| `Economics of Education` | Returns, markets, financing, and efficiency of education |
| `Welfare State` | Social protection and social policy |
| `Inequality` | Income, access, and opportunity inequality |
| `Institutions` | Formation, change, and persistence of institutions |
| `Development Economics` | Growth, historical development, and development economics |
| `Party Politics` | Parties, electoral competition, and representation |
| `Methodology` | Epistemology, research design, and methods |

## Layer C — Geographic scope (0 or 1 per post)

| Category | Description |
|---|---|
| `Brazil` | Brazil-focused work |
| `Latin America` | Latin America-focused work |
| `Comparative` | Multi-country work without a defined regional focus |

---

## Governance rules

- Posts **may** have multiple categories, but the **total number of distinct categories must remain small**.
- All concept-level terms (e.g., `causal-inference`, `regression-models`, `affirmative-action`) go in `tags`, not `categories`.
- No category names in Portuguese or mixed-language.
- Categories are Title Case; tags are kebab-case.
